# Setup Guide

## Windows

```bash
# Create virtual environment
python -m venv venv

# Activate virtual environment
venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run the notebook
jupyter notebook preproccess__feature_extraction.ipynb
```

---

## macOS

```bash
# Create virtual environment
```
